# 🖐️ AI Virtual Mouse

An AI-powered virtual mouse that uses **OpenCV**, **MediaPipe**, and **PyAutoGUI** to control the cursor with hand gestures.

## ✨ Features

- 🖱️ Cursor movement
- 👆 Left click
- 👉 Right click
- 👆👆 Double click
- 📸 Screenshot capture
- ⚡ Real-time hand tracking

## 🛠️ Tech Stack

- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- Pynput

## 🚀 Installation

```bash
git clone https://github.com/eesha95/AI-Virtual-Mouse.git
cd AI-Virtual-Mouse
pip install -r requirements.txt
```

## ▶️ Run

```bash
python main.py
```

## 🎯 Gesture Controls

| Gesture | Action |
|----------|--------|
| Index finger | Move cursor |
| Index finger bent | Left click |
| Middle finger bent | Right click |
| Index + Middle finger bent | Double click |
| Thumb close to index | Screenshot |
